# README

## Versions
* Ruby 2.3.1
* Rails 5.1.7
* Postgres 9.6.3

## Setup
* clone this repo
* run the following commands:
* bundle install
* rake db:create
* rake db:seed
* rake db:migrate

## Populate the Categories column of the Cafes table
* rake development:categorize

## How to run the test suite
* run "rspec"
